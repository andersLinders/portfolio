# Andrew Lindley — Portfolio

Product design portfolio landing page with persistent, animated project navigation.

## Develop

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes

- Desktop: hover a company to expand sub-items and crossfade the background.
- Mobile: tap to expand; sub-items stack below the parent.
- Typography currently uses **Outfit** as a stand-in for Nexa Pro from the Figma file. Drop licensed Nexa Pro files into `public/fonts` when ready and we can swap the face.
